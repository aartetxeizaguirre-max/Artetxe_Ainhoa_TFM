# Artetxe_Ainhoa_TFM
Pipeline en R para mi trabajo de fin de máster: Detección de firmas genómicas vinculadas con el avance metastásico en melanoma. Utiliza información de cBioPortal/TCGA (SNV, CNV, TMB) para identificar marcadores biológicos de progresión a través de análisis funcional y estadístico (GSEA/KEGG). Perspectiva centrada en la oncología de precisión
